Academic Experience

In my enrollment at the University of Guelph, I have gained considerable technical knowledge and had the opportunity to put them into practice. Below are technical areas I have practiced and/or gained knowledge in through my academics:

  • Data Structures - In completing the Data Structures course, I have learned about the many data structures for storing data (linked lists, hash-tables, stacks, etc.). The varying time and space complexities of each data structure and its methods plays a crucial role in determining how data should be stored in websites/applications. A hash-table works much better for accessing a website user's information from a database using their login, as opposed to a linked list which would have to iterate through all users to find your information.
  • Programming in C - The Introduction to Programming for Engineers course taught me the basics of programming in C. The covered concepts included loops and conditionals, arithmetic, logical, and bitwise operators, functions, etc. A year later taking the Data Structures course, I strengthened the previously mentioned skills in C, while also utilizing pointers to implement data stuctures such as linked lists, and memory management.
  • Object-Oriented Programming in Java - During the Winter 2023 semester, I took the Object-Oriented Programming course which taught the fundamental concepts of OOP. Through assignments done for the course, I learned about the conventions for set/get methods on objects, about inheritance and polymorphism being used to create like-objects with overridden and individualized traits and methods. Learning OOP taught me to abstract concepts when possible to object methods to provide more readable and modular code.
  • Probability and Statistics - As one of the most applicable areas of study in mathematics, I particularly enjoyed learning about probability and statistics. Some of the concepts I learned include probability theory, Bayes' Theorem, normal distributions and standard deviations, and linear regression. During this course, I used the knowledge I learned to write a paper that explains probability theory and Bayes' Theorem extensively, which can be found here. Learning about linear regression led to an interest in regression algorithms to create predictive models using Python data science libraries, such as Pandas and Scikit-learn.
  • Modelling Systems - After taking courses in linear algebra and system modelling, I have learned skills to model systems encompassing an incredibly wide variety (traffic flow, circuits, thermal systems, and more). Systems we were tested to mathematically model included systems of equations and matrices to group information together, differential equations to model how the state of a system changes, principle component analysis to determine which components of a system play the most significant role with them, and many more.
